## why this one is different from a shields.io badge

Shields.io badges are static strings someone typed once. This badge is **computed** — every time it's requested, the Worker:

1. Calls GitHub's public events endpoint for this account (no auth needed, no token to manage)
2. Buckets recent `PushEvent` timestamps into early bird / day shift / night owl / vampire hours
3. Renders an SVG inline, on the fly, with the result
4. Caches it for an hour so it's fast and doesn't abuse rate limits

If commit habits change, the badge changes — automatically, without touching this README ever again.
